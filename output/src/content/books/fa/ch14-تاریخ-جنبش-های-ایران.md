# خیزش دی‌ماه ۱۳۹۶

اولین جنبش ضدنظام (نه اصلاح‌طلبانه):

- شعار «اصلاح‌طلب، اصول‌گرا، دیگه تمومه ماجرا»

- گستره: شهرستان‌ها، طبقات پایین

- سرکوب سریع

# خیزش آبان ۱۳۹۸

::: enghelabbox
- علت: افزایش قیمت بنزین

- کشته: ۱۵۰۰+ (طبق رویترز)

- قطع اینترنت: ۱ هفته

- گستره: سراسری
:::

# جنبش مهسا ۱۴۰۱

- علت: کشته شدن مهسا امینی توسط گشت ارشاد

- شعار: «زن، زندگی، آزادی»

- ویژگی: رهبری زنان، نسل Z

- گستره: سراسری، همه اقشار

- مدت: چندین ماه

# روند تحول جنبش‌ها

::: olgoobox
1.  **از اصلاح به براندازی:** شعارها رادیکال‌تر شده

2.  **از نخبگان به توده:** شهرستان‌ها و طبقات پایین

3.  **از مردان به زنان:** نقش پیشتاز زنان

4.  **از ترس به شجاعت:** کاهش ترس از نظام
:::
