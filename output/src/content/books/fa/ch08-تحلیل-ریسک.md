# سناریوهای بدترین حالت

## سناریو ۱: سرکوب خونین

::: enghelabbox
نظام با استفاده از تمام ظرفیت سرکوب، اعتراضات را سرکوب می‌کند.

- **تلفات:** هزاران تا ده‌ها هزار

- **پیامد:** سکوت موقت، خشم انباشته

- **کاهش:** تاکتیک‌های پراکنده، اجتناب از تجمعات متمرکز
:::

## سناریو ۲: جنگ داخلی

::: enghelabbox
درگیری مسلحانه بین گروه‌های مختلف.

- **تلفات:** صدها هزار

- **پیامد:** ویرانی، مداخله خارجی

- **کاهش:** اجتناب از مسلح شدن، وحدت اپوزیسیون
:::

## سناریو ۳: تجزیه

::: enghelabbox
جدایی مناطق قومی.

- **احتمال:** پایین اما موجود

- **کاهش:** تضمین حقوق اقلیت‌ها، فدرالیسم
:::
