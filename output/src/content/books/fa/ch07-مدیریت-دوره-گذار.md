# نقش نیروهای مسلح

- **ارتش:** حفظ با پاکسازی فرماندهان + نظارت

- **سپاه:** انحلال یا ادغام تدریجی

- **پلیس:** بازسازی با اولویت

# عدالت انتقالی

::: olgoobox
1.  **حقیقت‌یابی:** کمیسیون حقیقت

2.  **محاکمه:** رهبران اصلی جنایات

3.  **جبران:** غرامت به قربانیان

4.  **اصلاحات:** تغییر نهادها
:::

::: noktebox
**تعادل:** عدالت باید با ثبات متوازن شود. عفو محدود برای عوامل رده پایین ممکن است ضروری باشد.
:::
